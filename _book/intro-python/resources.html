Python resources

Learn Python the Hard Way - An example-driven introduction to Python.

How to Think Like a Computer Scientist - Python Edition - A good intro if you're also interested in learning basic computer science concepts.

Python for Kids - A lively introduction to Python. The first half is particularly useful, you may wish to move on once the book focuses on creating games.

CodeAcademy: Python - An interactive tutorial for learning the basics of Python. (I recommend trying the books above first, as learning to use an IPython or IDE such as IDLE is an important part of the learning curve.) A good supplemental resource.
